# Freelancer Income Statistics Project

This project analyzes a freelancer income dataset using Python and Google Colab.

## Features
- Generate synthetic freelancer dataset (1,000 records)
- Display dataset information (head, tail, dtypes)
- Descriptive statistics (mean, median, std, skewness)
- Normal distribution visualization with KDE
- Z-score analysis and percentile ranking
- Correlation: Experience vs Hourly Rate
- Boxplot: Hourly Rate distribution by Skill

## Tech Stack
- Python
- Pandas, NumPy
- Seaborn, Matplotlib
- SciPy

## Dataset Variables
| Variable | Description |
|---|---|
| Skill | Freelancer's primary skill |
| Country | Country of operation |
| Experience_Years | Years of experience |
| Hourly_Rate_USD | Hourly rate in USD |
| Weekly_Hours | Hours worked per week |
| Client_Rating | Rating out of 5.0 |
| Remote | Remote work status |
| Monthly_Income_USD | Calculated monthly income |
